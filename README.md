# mcsmanager-web-unraid
An Unraid Community Apps template for MCSManager (MCSM) — a lightweight, open-source, web-based game server management panel. Supports Minecraft Java/Bedrock, Steam games, real-time logging, automated backups, and multi-server management. Template by the Unraid community.

This is an Unraid template for the web deployment. The daemon deployment is also required. See here: https://github.com/Billyboii/mcsmanager-daemon-unraid

This repo is based on the official MCSManager docker file: https://github.com/MCSManager/MCSManager?tab=readme-ov-file#docker-installation
