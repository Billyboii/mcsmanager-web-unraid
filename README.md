# mcsmanager-unraid
An Unraid Community Apps template for MCSManager (MCSM)

The web panel is a lightweight, open-source, web-based game server management panel. Supports Minecraft Java/Bedrock, Steam games, real-time logging, automated backups, and multi-server management.
The daemon is the backend component that manages and runs game server instances. Works alongside the MCSManager-Web template. Supports Minecraft Java/Bedrock and most Steam game servers.

This is an Unraid template made by the Unraid community for the web and daemon deployment. Both are required in order to use the tool fully.
This template is not currently associated with any members of the MCSM development team.

This repo is based on the official MCSManager docker file: https://github.com/MCSManager/MCSManager?tab=readme-ov-file#docker-installation
